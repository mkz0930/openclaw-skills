首先，感谢您花时间做出贡献！🎉

## 如何贡献

1. **Fork 仓库**
2. **创建分支**：`git checkout -b feature/your-feature`
3. **进行修改**
4. **运行测试**：`python3 /tmp/test_amazon.py`
5. **提交**：`git commit -m '添加一些功能'`
6. **推送**：`git push origin feature/your-feature`
7. **创建 Pull Request**

## 开发环境设置

安装说明请参见 [README-zh.md](README-zh.md)。

## 代码风格

- Python 代码遵循 PEP 8
- 使用一致的命名约定
- 复杂逻辑添加注释
