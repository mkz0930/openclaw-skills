# 🚀 OpenClaw 技能库

[EN](README.md) | [中文](README-zh.md)

OpenClaw 官方技能仓库 - Chrome Relay、亚马逊自动化、飞书集成

---

## 📋 可用技能

| 技能 | 描述 | GitHub |
|------|------|--------|
| **Amazon 卖家精灵 Chrome Relay** | 点击卖家精灵按钮并通过 Python WebSocket 提取数据 | https://github.com/mkz0930/amazon-seller-sprite-chrome-relay |
| **Chrome Relay 浏览器控制** | 完整的 Chrome 扩展中继自动化 | https://github.com/mkz0930/clawd-workspace/tree/master/skills/chrome-relay-browser-control |
| **飞书文档编辑** | 自动编辑飞书文档 | |
| **本地技能创建器** | 创建、验证、发布 AgentSkill | |

---

## 🚀 安装技能

```bash
# 克隆技能仓库
git clone https://github.com/mkz0930/amazon-seller-sprite-chrome-relay.git
cd amazon-seller-sprite-chrome-relay

# 运行安装脚本
./install.sh
```
