# Contributing to OpenClaw Skills

[EN](CONTRIBUTING.md) | [中文](CONTRIBUTING-zh.md)

First off, thanks for taking the time to contribute! 🎉

## How to Contribute

1. **Fork the repository**
2. **Create a branch**: `git checkout -b feature/your-feature`
3. **Make your changes**
4. **Run tests**: `python3 /tmp/test_amazon.py`
5. **Commit**: `git commit -m 'Add some feature'`
6. **Push**: `git push origin feature/your-feature`
7. **Open a Pull Request**

## Development Setup

See [README.md](README.md) for installation instructions.

## Code Style

- Follow PEP 8 for Python
- Use consistent naming conventions
- Add comments for complex logic

## Report Issues

Use the [GitHub Issues](https://github.com/mkz0930/amazon-seller-sprite-chrome-relay/issues) to report bugs or suggest features.
